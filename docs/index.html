目前看到的问题

What the Errors Suggest

  • 有些错误像是真正的计算或关系理解失败。
  • 有些输出包含多个数字和等式，last-number parser 可能取错数字。
  • weighted aggregation 题目可能诱导模型把两个数相加，而不是按权重计算。
  • Some failures look like genuine calculation or relation-understanding errors.
  • Some generations contain multiple numbers and equations, so the last-number parser may pick the wrong value.
  • Weighted aggregation prompts may lead the model to sum values instead of applying weights.

下一步

Next Step

先人工复核 20-30 条真实错误样例，判断哪些错误适合用 SFT 数据修复，哪些应先处理 prompt 或 parser。只有错误画像被确认是稳定模型弱点后，才进入 selection bias audit 和 LoRA 对比。

Manually review 20-30 real error examples first. Decide which failures are suitable for SFT data repair and which should be treated as prompt or parser issues before moving to selection-bias audits and LoRA comparisons.
